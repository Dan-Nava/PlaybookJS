PlaybookJS
Simplify the process of creating playbook-esque visuals and animations
Getting Started:

Note that this library utilizes JQuery, as such it must be acquired first.

On your webpage you will need both a JQuery script and a script for Playbook in this order:

<script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

<script defer type="text/javascript" src='js/Playbook.js'></script>

To setup a simple scenario of a Play, Field, and Token with a 1 branch path see the code below:

const play = new Play();
const field = new Field(play);
field.createField($('#example-one-container')[0], 'fID', '50em', '30em', '0.5em solid black', 'green');

const t = new Token(play);
t.createTokenVisual('blue', 13 + 'em', 18 + 'em', '4em', '4em', 'circle');

let joint = t.path.createPathJoint();
joint.breakPoint.createBPVisual('rgb(0,0,0, 0.4)', 8 + 'em', 2 + 'em', '4em', '4em', 'circle');
joint.branch.createBranchVisual('black', '12px', '12px');

//The animation can then be run by using:
play.runAnimations(500);

Token Object API

The central object in the library, all functionality is built to interact/manipulate this object.

Attributes: {Play play, Path path, Object animationInfo = { Int numOfAnimations, Int animationsFinished}, String id}
Function Description Return
allowDrag(boolean enable) Enables or Disables ability to drag token around when clicking and holding over token. N/A
createTokenVisual( rgb color, string xpos, string ypos, string width, string height, string shape) Creates Token DOM element based on specifications. shape can be either 'circle' or 'square' N/A
setTokenVisual(Token token, string xpos, string ypos) Takes a created DOM element and sets it as the Token's DOM element representation. Custom DOM elements can be used here to have them function as a token. N/A
setTokenImage(string imageSrc, string size, string repeat, string position) Sets a background-image for the Token DOM element. Size, repeat and position potential values are the same as CSS image equivalents N/A
getDOMElement() Retrieve's the objects respective DOM element DOM Element
setTokenPosition(string x, string y) sets position of token's DOM element. N/A
deleteToken() Removes Token from associated Path and Play object references and deletes the Token's DOM element N/A
canAddJointFromHere() Returns true iff Token currently has no BreakPoints, and thus can have its path extended from it. Boolean
animateAll(any array, int value, int duration, int delay, int iterations, string easing) Animates Token across all Branches and BreakPoints in the given array. Duration and Delay are in milliseconds. N/A
currentAnimation() Returns currentAnimation Object that is affecting the Token's DOM element Animation object
runAnimation(int duration, int delay, int iterations, string easing, any array) Calls animateAll iff Token is the start of the animation (start of its Path). N/A
resetAnimation() Resets animation loop and repositions Token DOM element to its initial position N/A
